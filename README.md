# 📊 Projeto Final de Estatística

Este projeto apresenta uma análise estatística de dados de consumidores de um e-commerce de moda - dados retirados do Kaggle.

O objetivo é identificar padrões de comportamento de compra, analisar relações entre variáveis e gerar insights sobre o perfil dos consumidores.

## 🎯 Objetivo
- Realizar análise descritiva dos dados
- Explorar relações entre variáveis
- Aplicar testes estatísticos
- Identificar perfis de consumidores

## 🛠️ Tecnologias utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📁 Arquivo principal
- `ProjetoFinal_Estatistica_.ipynb`

## 📊 Análises realizadas
- Estatísticas descritivas
- Distribuição das variáveis
- Análise entre idade, gênero e categoria de compra
- Impacto de descontos e promoções
- Perfil de consumidores

## 🚀 Como executar
1. Clone o repositório
2. Abra o notebook no Jupyter ou Google Colab
3. Execute as células em sequência

## 📌 Resultado
O projeto busca mostrar como a estatística pode ser aplicada para entender padrões de consumo e apoiar tomadas de decisão.
